# Learning Centre (MERN Stack)

## Installation

1. Clone repository:

git clone https://github.com/yourusername/learning-centre.git

2. Go to project folder:

cd learning-centre

3. Install all dependencies:

npm run install-all

4. Start application:

npm run dev

Backend: http://localhost:5000  
Admin: http://localhost:5173  
Frontend: http://localhost:5174
